# Content Source Workflow

Create and edit draft/original content in this folder.
Only publish finalized files to `assets/resources/`.

## Suggested flow
1. Draft in a category folder here.
2. Review and finalize.
3. Export final file (PDF/HTML/ZIP).
4. Copy final file into `assets/resources/`.
5. Add its metadata entry to `data/library.json`.

## Folders
- newspapers
- web-books
- toolbox
- workshops
- downloads
- newsletter
- templates
