Use this matrix to build high-quality prompts in 3-5 minutes.

Tool 1: 3x3 Prompt Builder

  • Pick 3 families.
  • Write 1 starter each.
  • Add 1 follow-up each.

Tool 2: Stem Bank Sprint

  • Generate 12 stems in teams.
  • Tag each by family.
  • Sort by subject fit.

Tool 3: Prompt Stress Test

  • Can the question be answered with evidence?
  • Does it invite revision?
  • Does it avoid yes/no closure?
Dense use tip: build a shared question bank by unit and reuse stems across subjects with only noun changes.

Tool 5: Facilitation Troubleshooter

  • Problem: one student dominates
    Fix: timed turns + evidence token system
  • Problem: no one responds
    Fix: pair warm-up then whole group
  • Problem: opinions with no evidence
    Fix: "cite before claim" rule
  • Problem: shallow agreement
    Fix: mandatory counter-perspective round

Tool Set D: AI Socratic Studio

Practical AI workflows for questioning, reflection, and curriculum-level thinking.

Tool 15: AI Dialogue Loop

Claimstudent draft
ProbeAI question
Evidencestudent cites
CounterAI alternate
Revisestudent update
  • AI asks one question only at a time.
  • No final answers until reflection step.
  • Require at least one citation in revised claim.

Tool 16: Prompt Pack (Teacher)

Role: Socratic coach for Grade 9-12. Rules: - Ask exactly one question per turn. - Sequence: Clarify -> Assume -> Evidence -> Perspective -> Implication -> Reflect. - Do not provide final answers. - After each student response, ask for either evidence or revision. - End with: "Rewrite your claim in 3 sentences with one source."

Tool 17: AI Safety Filter

  • Fact-check external claims.
  • No personal student data.
  • Flag uncertainty explicitly.
  • Teacher validates final output.

Big Idea: Beyond Classroom

Big Idea: Leadership Lens
